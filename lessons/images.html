Activity

Meet some new and ask ‘Hey can I make a website about you?’. Find a TEDI London Student and write a basic website about them. Allow them to rest assured that you will not be publishing this website so their data is safe. Save this page as hello.html.

Format the website as follows (assuming your new friend’s name is Paulo):
